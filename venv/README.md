# 世界盃足球賽預測系統 (World Cup Prediction)

這是一個結合 Python、資料工程、機器學習與 Power BI 的足球分析平台專案。

## 🚀 開發環境與啟動指南 (Cheat Sheet)

### 1. 如何開啟專案

- 打開 VS Code。
- 點擊左上角 `File` -> `Open Folder`，選擇 `world_cup_prediction` 資料夾。
- 💡 **快捷密技**：打開終端機 (Terminal) 進入專案資料夾後，輸入 `code .` 即可一鍵用 VS Code 打開。
